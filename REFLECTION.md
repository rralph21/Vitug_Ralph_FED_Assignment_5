## Intro
This project connects to the City of Winnipeg Open Data API to display tree information.

## Mechanics
When a user type a tree's common name like Ash or Maple and clicks "Search", the program will show the top 100 largest trees that matches the name.

# List

- The number of each tree in the ranking
- The Tree's common name
- The diameter of the tree's trunk - Diameter at Breast Height (DBH)
